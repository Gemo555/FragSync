# Roadmap

## Phase 0: Manual Demo And Demand Validation

- Hand-place beats and clips.
- Render a few montages manually.
- Learn which timing patterns feel good for FPS creators.

## Phase 1: Command-Line Prototype

- Build the local CLI.
- Load manual or automatic beats.
- Detect candidate clip events with simple signals.
- Generate a timeline.
- Render a basic montage with FFmpeg.

## Phase 2: Interactive Web MVP

- Add upload/select workflow.
- Show detected beats and clip events.
- Let users adjust event-to-beat mapping.
- Render from the approved timeline.

## Phase 3: CS2 / Valorant Optimization

- Add game-specific HUD and kill-feed regions.
- Improve gunshot and hit confirmation detection.
- Add presets for common resolutions and UI layouts.

## Phase 4: Style Templates

- Add clean sync, slow impact, aggressive, and cinematic templates.
- Implement velocity ramps and simple effects.
- Let users save repeatable style settings.

## Phase 5: Long-Video Highlight Detection

- Process full match recordings.
- Find likely kill clips automatically.
- Rank events and assemble first-draft montage candidates.
